# 💻 Laboratório de Programação (C)

Repositório criado para armazenar atividades, exercícios e projetos desenvolvidos na disciplina de **Laboratório de Programação** do curso de **Engenharia de Software - UFC (Universidade Federal do Ceará)**.

---

## 📌 Objetivo

Este repositório tem como objetivo:

* Organizar os códigos feitos durante a disciplina
* Servir como material de estudo e revisão
* Acompanhar minha evolução na programação
* Criar um portfólio inicial para futuras oportunidades

---

## 🛠️ Tecnologias utilizadas

* Linguagem C
* Compilador GCC

---

## 📖 Conteúdos abordados

* Variáveis e tipos de dados
* Estruturas condicionais (if, else)
* Estruturas de repetição (for, while)
* Funções
* Structs
* Manipulação de strings
* Entrada e saída de dados

## 🚀 Como executar os códigos

### 1. Compilar

```bash
gcc nome_do_arquivo.c -o programa
```

### 2. Executar

```bash
./programa
```
